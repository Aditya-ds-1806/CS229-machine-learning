# A muddled muff's musings on Machine Learning

## Why this repo?

FOMO.

The AI world is moving ridiculously fast—new models, new tools, new "this changes everything" drops every other day. It's exciting, but also a bit overwhelming.

At some point I realized I don't really have a **solid understanding of what's going on underneath all this**. And that didn't sit well.

So this is me trying to fix that.

Not trying to predict the future.  
Not trying to chase every new shiny thing.  
Just trying to **build fundamentals properly**.

---

## What's in here?

I'm going through Andrew Ng's **CS229: Machine Learning** course and documenting things as I go.

Each topic has a Jupyter notebook with:

- Notes from the lecture (in my own words)
- Math (I'm trying to actually derive things, not just accept formulas)
- Some code when I feel like I need to "see" what's happening
- Occasional plots/visualizations

---

## How I'm approaching this

I'm trying to not rush this.

- Understand the math properly (or at least try to)
- Build intuition alongside it
- Write code where it helps things click

Basically: **less passive watching, more active struggling**

---

## Who is this for?

Honestly, mostly for me.

But if you're:

- Also going through CS229
- Or just revisiting ML fundamentals
- Or like seeing someone else struggle through derivations

you might find this useful.

---

## That’s it

This is me trying to be a student again.

If you’re doing something similar, welcome 🤝
